NOTE : Please click on the jewellery for trying it on a virtual mirror
NOTE : The 'tryonpage.html' accepts the following parameters :-
  imageurl  :  The URL of the Jewellery(Earring Only) Image to Virtually Try on.
  imagetype  :  Image type(EARRING).
  fullscreen  :  Can be 'true'/'false'. If set to true it will display the try on mirror in full screen. It's default value is set to 'true'.
  verbose  :  Can be 'true'/'false'. If set to true it will display additional info on mirror. It's default value is set to 'false' .
  EDT  :  Can be a value between 0 and 1. It is a threshold value which determines which all ears are visible based on the rotation calculated by the system. It's default value is set to 0.28 .
  EPT  :  It can be used to displace the earring images from calculated placement positions. It's default value is set to 5 .
  FTT  :  Can be a value between 0 and 1. It is used to set the face tilt that is acceptable. It's default value is set to 0.25 .
  JScale  :  Can be a value between 0 and 1. It is used to adjust the size of the jewellery image. It's default value is set to 0.5 .

Parameters having default values is optional.
